/*
 * Функція: hashPJW
 * --------------------
 * Обчислює геш-значення для даного рядка.
 *
 * Ця функція використовує алгоритм PJW (Peter J. Weinberger) для генерації геш-значення для рядка.
 * Цей алгоритм використовує бітові операції для створення гешу, який добре розподіляє рядки по геш-таблиці.
 *
 * str: Вхідний рядок, для якого потрібно обчислити геш-значення.
 *
 * повертає: Геш-значення вхідного рядка, обмежене розміром геш-таблиці.
 */

 /*
 * Функція: resizeHashTable
 * --------------------
 * Змінює розмір геш-таблиці на новий, збільшуючи його удвічі.
 *
 * Повертає нову геш-таблицю зі зміненим розміром.
 */

 /*
 * Функція: insert
 * --------------------
 * Додає новий елемент до геш-таблиці.
 *
 * key: Ключ елемента.
 * film: Назва фільму.
 * year: Рік випуску фільму.
 */

 /*
 * Функція: search
 * --------------------
 * Пошук елемента за ключем в геш-таблиці.
 *
 * key: Ключ елемента, який потрібно знайти.
 *
 * повертає: Вказівник на знайдений елемент або NULL, якщо елемент не знайдено.
 */

 /*
 * Функція: deleteD
 * --------------------
 * Видалення елемента з геш-таблиці за ключем.
 *
 * key: Ключ елемента, який потрібно видалити.
 */

 /*
 * Функція: loadDataFromFile
 * --------------------
 * Завантажує дані з файлу до геш-таблиці.
 *
 * filename: Ім'я файлу, з якого потрібно завантажити дані.
 */

 /*
 * Функція: printHashTable
 * --------------------
 * Виводить зміст геш-таблиці на екран.
 */
